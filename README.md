# OpenWrt-Action

本项目使用SuLingGG的 [Action-OpenWrt-Plus](https://github.com/SuLingGG/Action-OpenWrt-Plus) 项目:

特此感谢SuLingGG大佬十分方便的配置和工作流

## 支持设备
1. Raspberry Pi 3b/3b+
2. X86_64 通用软路由
3. 还在不断增加设备，敬请期待

注:

1. 项目默认关闭了IPV6，需要手动开启
2. 项目SuLingGG的模板，前几版暂时未修改，目前项目主仍在学习
